#### How to use

##### Server
```bash
$ git clone https://github.com/PeterPimentel/mirror-drawing
$ cd mirror-drawing/
$ cd server
$ npm install
$ npm start
```

##### Client
```bash
$ cd mirror-drawing/
$ cd client
$ npm install
$ npm start
```

Remenber of install and prepare the database
